# Description
 A mysql database and CMS-style Blog built using Model View Controller  paradigm. Built using MySQL2, Express, Sequelize,  Handlebars and dotenv.
# Installations
` express`
   `npm init -y`
   `express-handlebars`
   `npm i mysql`
   `npm i sql`
   `npm i mysql2`
   `express-session`
   `connect-session-sequelize`
